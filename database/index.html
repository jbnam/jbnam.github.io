Database for Twists

Let $E$ be an elliptic curve defined over $\mathbb{Q}$ of conductor $N$ and $\chi$ be a Dirichlet character. Then, the $L$-function of an elliptic curve $E$ twisted by $\chi$ is defined by the following Dirichlet series for $\text{Re}(s) > 3/2$: $$L(E, s, \chi) := \sum_{n \ge 1}\frac{\chi(n)a_n}{n^s} = \prod_{p \nmid N}\Big(1 - \frac{\chi(p)a_p}{p^s} + \frac{\chi^2(p)}{p^{2s-1}}\Big)^{-1}\prod_{p \mid N}\Big(1 - \frac{\chi(p)a_p}{p^s}\Big)^{-1}$$ where $a_p$ is the traces of Frobenius of $E$ if $p \nmid N$ and $0, \pm 1$ depending on the reduction type of $E$ modulo $p$ otherwise. It is well-known that it can be analytically continued to $\mathbb{C}$ and satisfies some functional equation which relates $s$ to $2-s$, so that the critical strip is $\{s \in \mathbb{C} \mid 1/2 < \text{Re}(s) < 3/2\}$.

The database contains some arithmetic values including $L(E, 1, \chi)$ and their algebraic parts for the families of primitive Dirichlet characters of order $k = 3, 5, 6, 7, 13$ whose conductors are coprime to $N$ and up to $3\cdot 10^6$ for $k = 3, 5, 7, 13$ and $10^6$ for $k = 6$. The values of $L(E, 1, \chi)$ in Database are computed by a program written by C using FLINT and CUDA for high performance computings with a large number of $a_n$’s precomputed by PARI/GP. Moreover, their algebraic values are computed by SageMath.

  1. For the overview of computations of $L(E, 1, \chi)$ and access to the data, go to Central $L$-values.

  2. For the overview of their algebraic values and access to the data, go to Algebraic Values.

HOW TO READ THE DATA FILES: The data file type is the Sage object file (i.e. .sobj). Thus, one may need to run SageMath to read it.
